islands.futile.me ↗ [WIP]

A public collective garden. You will be able to claim "islands" and attach a message to them, then other users can plant flowers on your island to increase its lifespan. Islands all eventually fall back into the ocean, leaving room for new ones to appear.

Gatherbun ↗

NextJS app that uses the Prisma ORM with a PostgreSQL database, all containerized with docker and hosted on VPS with CI/CD using GH Actions and using Stripe as a payment processor.

futile.me ↗

futile.me is a minimal writing platform built with the TipTap editor, custom CSS, and classic username + password auth via iron-session. It skips email in favor of a 12-word mnemonic recovery system for security and ease of use.

Crypto Hoards ↗

Crypto Hoards was a Solana Web3-connected web app that connected to your solana wallet and displayed your media NFTs. The code is available on my GitHub as it is now offline, was a project to dip my toes into web3.
